Telegram Bot for Invoice Photo Upload to Yandex Disk

Setup:
1. Install dependencies: pip install -r requirements.txt
2. Set environment variables:
   - TELEGRAM_BOT_TOKEN: Your Telegram bot token from BotFather
   - YANDEX_DISK_TOKEN: Your Yandex Disk OAuth token
3. Run: python bot.py

The bot guides users through city selection, point (for Saint Petersburg), date, and photo upload.
Photos are uploaded to Yandex Disk in /city/point/date/ folders.# vaffel_disk_bot
